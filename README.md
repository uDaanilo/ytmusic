## ✔️ Features

- ⏯ Miniplayer
- 🚫 Adblock
- ✨ Discord Rich Prensence

## 🚧 Development

### ⚒️ Quick start

1. Cloning repository
```sh
$ git clone https://github.com/uDaanilo/ytmusic.git
```
2. Downloading dependencies
```sh
$ npm i
$ yarn
```
3. Starting project
```sh
$ npm run dev
$ yarn dev
```

### 🧩 Building project

```sh
$ npm run build
$ yarn build
```
